Sie werden drei unterschiedliche Tests durchfuehren. Jeder Test wird 30x durchgefuehrt.
3. Test:
Anleitung: Am Bildschirm werden die Namen von Tieren angezeigt.
  • Das Tier ist kein Saeugetier. Druecken Sie ' f '.
  • Das Tier ist ein Saeugetier: Druecken Sie ' t '
  • Wird die falsche Taste gedrueckt, zaehlt das als Fehler.

Scoring-System:

  • Sie erhalten Pluspunkte fuer richtige Antworten und Maluspunkte fuer falsche Antworten.
  • Es gibt einen Speed-Bonus. Bonus-Punkte fuer schnelle Antworten und Maluspunkte fuer zu langsame Antworten.
  • Wenn Sie mehrere richtige Antworten in Folge abgeben, erhalten sie einen hoeheren Punktemultiplikator. Der Multiplikator wird erhoeht nach 2, 4, 7, 11, ... Antworten./li>
